🧠 AI Image Classifier

A modern desktop image classification application built with PyTorch, ResNet18, and a custom Tkinter dark UI. The application supports both single-image and batch-folder classification with real-time progress tracking, live previews, and detailed prediction logs.

🚀 Highlights
🖼️ Single image classification
📂 Batch folder classification
🧠 Pretrained ResNet18 neural network
🎯 1000 ImageNet object categories
📊 Top-5 predictions with confidence scores
⚡ Multi-threaded processing (responsive UI)
⏳ Real-time ETA estimation
📈 Live progress tracking
🖥️ Dynamic image preview
🧾 Activity logging panel
🌙 Modern dark-themed interface
🔄 Automatic ImageNet label download & caching
🖼️ Application Features
Single Image Classification

Select an image and instantly receive the top predictions generated by the ResNet18 model.

Batch Folder Processing

Classify hundreds of images inside a folder while monitoring:

Progress percentage
Estimated remaining time (ETA)
Current image preview
Live classification results
Activity Logging

Every classification operation is recorded in a built-in log console for easy monitoring and debugging.

🧠 AI Model
Specification	Value
Model	ResNet18
