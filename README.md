🧠 AI Image Classifier (Desktop App)

A modern desktop application for image classification using PyTorch and a pretrained ResNet18 model, with a clean and responsive Tkinter UI.

✨ Features
🖼️ Single image classification
📂 Batch folder classification
🧠 1000 ImageNet classes
📊 Top-K predictions with confidence scores
⚡ Real-time progress bar + ETA
🧾 Activity log panel
🎨 Clean modern dark UI
🖥️ Live image preview
🧠 Model Details
Architecture: ResNet18
Dataset: ImageNet (1000 classes)
Framework: PyTorch
Runs on: CPU (default)
🖥️ How to Use
🔹 Classify Single Image
Click Browse
Select an image
Click Classify
🔹 Classify Folder
Click Choose...
Select a folder
Click Classify Folder
